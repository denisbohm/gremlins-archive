If you have your own web pages, Gremlins in the Garage encourages you to create a link to us!

To create a graphical link for you can use this html snippet: 

<a href="http://www.gremlins.com"><img
src="http://www.gremlins.com/garage/gremlins_icon.gif"
width=160 height=50
alt="Gremlins in the Garage!"></a>

To create a textual link for Gremlins in the Garage! you can use this html snippet: 

<a href="http://www.gremlins.com">Gremlins in the Garage!</a>
