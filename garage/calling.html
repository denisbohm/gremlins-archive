International Dialing Instructions from the US

Dial in sequence:
1. 011 - the International Access Code.
2. The country code.
3. The city code.
4. The local telephone number.
5. The '#' button - if using a touch tone phone (this step saves additional time).

For example, to call Zotz you would dial:

1. 011 - the International Access Code.
2. 52 - The country code for Mexico.
3. 5 - The city code for Mexico City.
4. 202-1544 - The local telephone number for Zotz.
5. The '#' button - if using a touch tone phone (this step saves additional time).
So the final calling sequence for Zotz is: 011 52 5 202-1544 #.
