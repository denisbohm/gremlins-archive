ust become a gremlins member and want some background on how to add new stuff to the gremlins web site? You've landed in the right spot!

What do you want to do?

Add a buildup to the gallery section.

  1. Prepare a full sized image and an icon image on your computer according to the instructions on the picture page.
  2. Logon to the gremlins web site.
  3. Put your images on the gremlins server.
  4. Note or add the kit information.
  5. Add the buildup.

Add a new kit announcement to bites section.

  1. Prepare a bites image on your computer according to the instructions on the picture page.
  2. Logon to the gremlins web site.
  3. Put your image on the gremlins server.
  4. Note or add the kit information.
  5. Add the bites announcement.

Add a kit review to the articles section.

  1. Prepare the review text on your computer.
  2. Prepare a review image on your computer according to the instructions on the picture page.
  3. Logon to the gremlins web site.
  4. Put your image on the gremlins server.
  5. Note or add the kit information.
  6. Add the kit review.

Update your contact information

  1. Logon to the gremlins web site.
  2. Use the contact page to search for your name.
  3. Click on the "Edit" button.
  4. Change the information on the form.
  5. Click "Update".

Note kit information

  1. Logon to the gremlins web site.
  2. Use the kit page to search for the kit.
  3. If the kit is found click on the kit name to view the details.
  4. Click on the "Note" button.

Add kit information

  1. Logon to the gremlins web site.
  2. Use the kit page to search for the kit.
  3. If the kit is found click do not add the kit information again - just note the kit.
  4. If the kit is not found then note or add the sculptor and producer.
  5. Go to the kit page and click on the "Add" button.
  6. Enter the kit info and click on the "Insert" button.

Buttons

Once you have been given rights to add content to the gremlins web site and you Logon you will see some extra buttons when you view gallery entries, kits, contacts, reviews, etc.

ButtonDescription
The Add button will take you to a form to add information to the web site.
The Edit button will take you to a form to update the information on a web page.
The Update button applies the changes you have made in a form to the gremlins web site.
The Insert button adds the information in a form to the gremlins web site. Please, always search to make sure that you do not insert duplicate information into the database!
The Note button will add an item to your favorites list so that it will appear on the various forms.
The Toss button will remove an item from your favorites list so that it will not appear on the various forms.

Favorites

Some fields on the various forms give you a limited set of choices - these choices are called favorites. To add an item to the set of choices you search for the item and view it then press the Note button.

Getting Started

The first thing you should do after you logon for the first time is to use the contact page to search for yourself, then view your information, and click the Note button. That way your name will show up as one of the choices for the painter when you use the buildup page to add one of your buildups to the gallery. The resources section has links to all the pages that can be used to search and add information - so use the resource page as your starting point.

If you run into any trouble please join the members mailing list and ask questions.
